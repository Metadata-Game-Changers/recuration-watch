See how complete your repository’s metadata are — then improve them.

Six free, browser-based tools from Metadata Game Changers that measure the completeness, connectivity, and curation history of any DataCite repository against live data — and turn what they find into action, matching the people and organizations in your metadata to the ORCID and ROR identifiers that connect research together.

100% in your browser — nothing uploaded, no login Live data from DataCite, ROR, and ORCID APIs Works with any of DataCite’s 3,500+ repositories Open — CC BY-NC 4.0, source on GitHub
Measure · FAIR and Projects

Metadata Completeness

Measures metadata completeness using the MGC FAIR and Project use cases — one concept at a time.

  • Concept-by-concept completeness with drill-downs to see real values
  • HTML / PDF / JSON reports for sharing and future analysis
  • Filter by resource type, query, or random sample
Open Metadata Completeness →
Measure · Identifiers

Metadata Connectivity

Measures how well creators, contributors, funders, publishers, and rights carry the identifiers that connect them — and finds quick wins.

  • ORCID, ROR, funder, publisher, and rights coverage at a glance
  • Drill-downs list every unidentified name — quick wins first
  • One click sends them to the ROR / ORCID Retrievers with tabular data reports
Open Metadata Connectivity →
Watch · Curation

Re-Curation Watch

Browses a repository’s most recent updates — or the full curation history of a single DOI — to see re-curation as it happens.

  • Version-by-version changes for any DataCite DOI
  • Spot metadata improvements (and regressions) over time
  • Make your invisible metadata curation work visible
Open Re-Curation Watch →
Watch · Activity

Repository Activity

Explore repository complete curation activity over time — creates, updates, and publishes — to demonstrate and understand active collection curation.

  • Activity timelines for any DataCite repository
  • Compare curation effort across collections and years
Open Repository Activity →
Retrieve · Organizations

ROR Retriever

Matches affiliation, funder, and publisher names to ROR identifiers using the same affiliation-matching strategies DataCite, Crossref, and ROR use.

  • Careful algorithmic matching first, full query search as fallback
  • Color-coded confidence with full candidate details
  • CSV results ready for analysis and re-curation
Open ROR Retriever →
Retrieve · People

ORCID Retriever

Finds ORCID iDs for lists of people, showing each candidate’s most recent employment and journal so the right person is easy to pick.

  • Handles “First Last” and “Last, First” name forms
  • Built-in disambiguation for common names
  • CSV results ready for analysis and re-curation
Open ORCID Retriever →
The tools work as a pipeline. Measure a repository’s completeness and connectivity → drill down to the specific people and organizations missing identifiers → retrieve their RORs and ORCIDs with one click → re-curate your metadata and watch the completeness improve. Every hand-off carries its context, so you always know which dataset you are improving.
