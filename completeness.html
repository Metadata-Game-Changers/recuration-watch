About: This tool measures DataCite metadata completeness for a repository using the Metadata Game Changers FAIR spirals — Text, Identifiers, Connections, and Contacts. Each concept's score is the share of records that contain it; the Total index is the concept-weighted average across all four spirals. All scoring runs in your browser against live DataCite data — nothing is uploaded or stored.
Troubleshooting: if the scoring engine can't load from a CDN, download jq.asm.bundle.js from https://unpkg.com/jq-web@0.5.1/jq.asm.bundle.js, place it beside this file, and point the loader's first source at ./jq.asm.bundle.js.
