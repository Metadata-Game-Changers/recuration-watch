About: This tool measures DataCite metadata completeness for a repository using the Metadata Game Changers FAIR use cases — Text, Identifiers, Connections, and Contacts. Each concept's score is the share of records that contain it; the Total index is the concept-weighted average across these four use cases. Five additional use cases — DataCite Relations (41 relation types), DataCite Contributors (20 contributor types), and Project Team Members, Project Items, and Project Relations — are also scored and viewable, but they are reported separately and do not contribute to the Total index. All scoring runs in your browser against live DataCite data — nothing is uploaded or stored.
Troubleshooting: if the scoring engine can't load from a CDN, download jq.asm.bundle.js from https://unpkg.com/jq-web@0.5.1/jq.asm.bundle.js, place it beside this file, and point the loader's first source at ./jq.asm.bundle.js.
