Large-repository guard: the count is read first; if it exceeds 10,000 events the tool loads only the most recent capped set and labels the result partial — it never pulls all activities for a giant repository (DataCite page-number pagination tops out at 10,000; beyond that needs cursor pagination, left out of this prototype).
