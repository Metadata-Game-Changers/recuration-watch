Find ORCID iDs for peopleprototype

Paste person names (one per line, “First Last” or “Last, First”). Each is searched in the public ORCID registry; every candidate iD is shown with its most recent employment and most recent journal to help pick the right person. Everything runs in your browser; nothing is uploaded or stored.

ORCID matches

Names are searched as family-name AND given-names in the public ORCID registry — the same search MGC’s getORCIDs.py uses. Green = exactly one ORCID matches the name, amber = several match (one row per candidate — use the employment and journal columns to pick), red = none. ORCID profile data appears only if the person made it public. Each iD links to its orcid.org profile.
